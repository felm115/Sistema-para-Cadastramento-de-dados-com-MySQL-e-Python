## Sistema de Cadastro de Alunos

## Projeto

- Usei Python (Tkinter) para fazer a interface gráfica para o cadastramento dos alunos e MySQL para armazenamento dos dados. 
- A partir disso, criei um sistema que permite realizar o gerenciamento completo de alunos por meio das operações de cadastro, consulta, pesquisa, exclusão e exportação de dados para planilhas Excel.

## Tecnologias Utilizadas

- Python, Tkinter, MySQL, MySQL Connector e OpenPyXL


## Objetivo do Projeto

- Criação de um banco de dados com MySQL
- Integração entre Python e MySQL
- Se necessário, exportar os dados para Excel para uma análise melhor
- Criação de interfaces gráficas profissionais.


## Instruções

- Instalação

pip install -r requirements.txt

- Configuração MySQL

No arquivo main.py (DB_CONFIG), altere:
password: "SUA_SENHA_AQUI"  para a senha do seu MySQL.

## Executar

python main.py no terminal
